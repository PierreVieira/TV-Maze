# 🧩 Components - Componentes Reutilizáveis

Este diretório contém todos os **componentes reutilizáveis** construídos com Jetpack Compose que podem ser utilizados em múltiplas telas do aplicativo TV Maze.

## 🎯 Objetivo

Os componentes garantem:
- ✅ **Reutilização** de código UI
- ✅ **Consistência** visual
- ✅ **Manutenibilidade** centralizada
- ✅ **Testabilidade** isolada
- ✅ **Performance** otimizada

## 📋 Componentes Disponíveis

### 🔘 icon_button/
**Botões com Ícones** - Botões customizados com ícones e diferentes estilos.

### 🎬 media_item_card/
**Cards de Mídia** - Cards responsivos para exibir séries de TV.

### 🖼️ picture/
**Componente de Imagem** - Componente avançado para carregamento de imagens.

### ✨ shimmer/
**Efeito Shimmer** - Efeitos de loading para melhorar a percepção de performance.

### 📏 spacer/
**Espaçadores** - Espaçadores personalizados para layout.

### ⭐ stars/
**Avaliação em Estrelas** - Componente de avaliação visual.

## 🔧 Características Técnicas

### 🎨 **Design System**
- **Material 3** - Seguindo o design system moderno
- **Consistent spacing** - Espaçamentos padronizados
- **Color scheme** - Cores consistentes
- **Typography** - Tipografia padronizada

### 📱 **Responsividade**
- **Adaptive layouts** - Layouts que se adaptam ao tamanho da tela
- **Tablet support** - Suporte a tablets
- **Desktop support** - Suporte a desktop
- **Orientation changes** - Suporte a mudanças de orientação

## 📊 Componentes em Detalhes

### 🎬 **MediaItemCard**
Card principal para exibir séries de TV com poster, título, informações e botão de favorito integrado.

### 🖼️ **AsyncPicture**
Componente de carregamento assíncrono de imagens com placeholder, error state e cache automático.

### ⭐ **StarsRating**
Avaliação visual em estrelas com diferentes tamanhos, texto opcional e animações.

## 🔗 Dependências

- **Jetpack Compose** - Framework de UI
- **Coil** - Carregamento de imagens
- **Material 3** - Design system
- **Core Model** - Modelos de domínio
- **UI Theme** - Temas e cores
